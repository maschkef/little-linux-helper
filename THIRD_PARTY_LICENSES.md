## License Texts

### Go Dependencies

Full license texts for Go dependencies are available in the `licenses/go/` directory. Each module has its own subdirectory containing the complete license file(s).

### Node Dependencies

Full license texts for Node dependencies are available in the installed `licenses/node/` directory after running `npm install`. Each package contains its own `LICENSE` or similar file. A JSON summary of all Node package licenses is also available in `licenses/node/licenses.json`.

### Retrieving License Texts

To regenerate or update the license information:

**Go Dependencies:**
```bash
cd gui
go install github.com/google/go-licenses@latest
go-licenses save . --save_path=../licenses/go --force
```

**Node Dependencies:**
```bash
cd gui/web
npx --yes license-report --output=json --only=prod > ../../licenses/node/licenses.json
```
